# @taucad/units

Units of measurement and conversions for the Tau CAD platform.

## Subpaths

- `@taucad/units` — core unit types
- `@taucad/units/constants` — unit constants
- `@taucad/units/converter` — unit converters
- `@taucad/units/parser` — unit parsing

Part of the [Tau](https://github.com/taucad/tau) monorepo.
